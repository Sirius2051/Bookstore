# Bookstore

### Installation

1. Install Django
   ```sh
   pip install django
   ```
2. Install Mysqlclient
   ```sh
   pip install mysqlclient
   ```
3. Install Django Widget Tweaks
   ```sh
   pip install django-widget-tweaks
   ```
4. Run migrations
   ```sh
   python manage.py migrate
   ```
5. Running the Local Development Server
   ```sh
   python manage.py runserver
   ```
